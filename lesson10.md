* 비트 : >>, <<, &, |, ^, ~
(진법을 알아야 이해하기 쉬움)
```C#
int x = 61;  
//0000 0000 0000 0000 0000 0000 0011 1101(2)
//  0    0    0    0    0    0    3   D  (16)
int y = 55; 
//0000 0000 0000 0000 0000 0000 0011 0111(2)
//  0    0    0    0    0    0    3    7  (16)

Console.WriteLine(x << 2);  //x에 들어 있는 값을 2진수로 바꾸고 비트를 왼쪽으로 2번 이동시킨다.
//앞에 '00'이 빠지고 오른쪽 끝에 '00'이 붙음 0000....0011 1101 00(2)
//128 + 64 + 32 + 16 + 4 = 244(10)
//<< 2를 하면 10진수는 4배한 값이 나옴


Console.WriteLine(x >> 2);  //우측으로 2번 이동
//우측으로 하면 2로 나누는 것과 같은데 소수가 나오면 소수점은 잘림(정수만 남음)
//0000.... 0011 11(2)  -> 15(10)   한번씩 밀때마다 나누기 2한 효과를 얻음

//formatting
Console.WriteLine("{0:X8} {1:X8}",x ,y);   //16진수 8자리로 출력 , X8->8자리 표기한다는 의미

```
